> ## 命令
> ### 参数说明
> 必选：`{...}`
>
> 可选：`[...]`
>
> ### 重载配置
>
> ```shell
> /dragon-bloom:d-bloom reload
> ```
>
> ### 生成粒子
> #### type: `location`, `follow`
> #### particle: `String`
> #### life: `int`
> #### rotation: `String`
> #### translate: `String`
>
> ```shell
> /dragon-bloom:d-bloom location {particle} [life] [rotation] [translate]
> ```
